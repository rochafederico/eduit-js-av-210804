identificador de recursos uniforme

uniform resource id

[esquema]://[autiridad]/[ruta]/?[parametro]=[valor]#[anclas]

Esquema

  • http/https
  • ftp
  • mailto
  • tel
  • data
  • irc

Segun el equema la URI puede tener distinto formato

Autoridad

Segun el caso podemos tener las 3 partes o solo una o dos

  • userinfo (opciones)
    usuario:clave@
    usuario@
  • anfitrion
    ip o domino
  • puerto
    :3000

ruta

Jerarquizar los recursos

  • /cursos
  • /cursos/obtener-totos
  • /cursos/alumnos

Query o consulta

si necesitamos pasar parametros al servidor usamos "clave=valor"

  • ?nombreparametro=123
  • ?nombreparametro=123&otroparametro=valor

Fragmento, hash o ancla

Contine un id de un elemento dentro del DOM
